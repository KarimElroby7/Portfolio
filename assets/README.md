# Assets

Drop your images here.

## Required

- **`profile.jpg`** — your hero photo. Used in the hero section as a circular framed image.
  - **Recommended:** square crop, ~800×800 px or larger, professional headshot.
  - The site renders a clean gradient "KE" placeholder if this file is missing.

## Optional

- `og.png` — 1200×630 social preview image. Reference it in `index.html`'s `<meta property="og:image">` once added.
- `favicon.ico` / `favicon.svg` — browser tab icon. Add a `<link rel="icon" href="assets/favicon.svg">` to `<head>` after dropping it in.
